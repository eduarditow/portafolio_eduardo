Primer Parcial

Presentacion de parcial

Marzo 13, 2018

Aplicando las metodologías para la creación de un sitio web y los principios de diseño realizar un diseño exclusivo de acuerdo al tema asignado al inicio de la materia, se le solicita lo siguiente:

1. Tener un objetivo claro del Sitio

2. Realizar un boceto a mano alzada

3. Definir una estructura

4. Proponer un diseño exclusivo para su sitio web

a. Contraste

b. Proximidad

c. Alineación

d. Repetición

5. El sitio deberá contener al menos 4 páginas web todas accesibles entre ellas, además deberá de distribuir el contenido de forma coherente..
